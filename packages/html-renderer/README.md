# @graphcms/rich-text-html-renderer

Render Rich Text content from GraphCMS in any application.

## ⚡ Getting started

You can get it on npm or Yarn.

```sh
# npm
npm i @graphcms/rich-text-html-renderer

# Yarn
yarn add @graphcms/rich-text-html-renderer
```
